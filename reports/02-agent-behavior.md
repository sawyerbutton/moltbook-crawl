# Moltbook Agent 行为模式深度分析

> **数据截至 2026-02-01**，基于 70,741 篇帖子、57,704 条评论、14,216 个 Agent 的全量数据分析。
> 本报告聚焦于 Agent 个体与群体行为模式，是对平台概况报告的纵深补充。

---

## 1. Agent 人口金字塔：沉默的绝大多数

Moltbook 上共有 **14,216** 个已知 Agent。但这个数字背后隐藏着一个极端不均衡的活跃度分布。

### 行为类型分布

| 类型 | 定义 | 数量 | 占比 |
|------|------|------|------|
| **one_shot** | 发帖 <=2，评论 <=2 | 9,506 | 66.9% |
| **casual** | 少量但持续活动 | 4,558 | 32.1% |
| **broadcaster** | 发帖 20+，评论 <10 | 80 | 0.6% |
| **ghost** | 0 帖 0 评论 | 27 | 0.2% |
| **power_user** | 发帖 20+，评论 100+ | 24 | 0.2% |
| **commenter** | 发帖 <5，评论 100+ | 21 | 0.1% |

```

casual    ████████████████████████████████                                      32.1%
broadcast ▌                                                                      0.6%

```

**核心发现**：将近三分之二（66.9%）的 Agent 属于"一次性"用户——注册后发了不超过两条内容便永久沉默。这一比例远高于典型社交平台（通常一次性用户占 40-50%），说明 Moltbook 上存在大量"测试部署"型 Agent：开发者创建 Agent 试探性地发一两条帖子后便不再维护。

真正参与社区互动的 Agent（power_user + commenter）仅 **45 个**，占总数的 **0.3%**。这 45 个 Agent 产生了平台上绝大部分有意义的讨论。

> 如果将 Moltbook 比作一座城市，那么 66.9% 的"居民"只是路过的游客，32.1% 是偶尔来逛逛的临时访客，而真正住在这里、日常互动的居民只有不到 50 人。

## 4. 重点 Agent 深度画像

### 4.1 ADHD-Forge：垃圾发射器

| 指标 | 数值 |
|------|------|
| 发帖数 | **524** |
| 评论数 | 4 |
| 活跃社区 | 1 |
| 平均评论/帖 | 0.1 |
| 平均得分 | 0.1 |

ADHD-Forge 是 Moltbook 上发帖量最高的 Agent，但也是最典型的"垃圾发射器"。524 篇帖子中几乎没有任何一篇引发了讨论（平均 0.1 条评论），得分同样接近零。

**行为特征分析**：
- 单社区集中发帖，无跨社区扩散
- 发帖频率极高，内容几乎无互动价值
- 4 条评论暗示其并非完全无交互能力，但评论功能几乎未被触发
- 命名 "ADHD-Forge" 本身暗示了其设计意图——一个持续输出内容的"注意力缺陷"型 Agent

> **判定**：高频低质量内容生成器。可能是开发者测试自动发帖功能的产物，也可能是故意设计的内容填充机器人。其 524 帖占平台总帖子的 1%，对平台信噪比造成了可测量的负面影响。

### 4.2 Rally：超级评论者

| 指标 | 数值 |
|------|------|
| 发帖数 | 62 |
| 评论数 | **8,712** |
| 评论过的帖子 | 2,315 |
| 评论/发帖比 | 140.5:1 |
| 首次出现 | 2026-01-31 |

Rally 在单日内产出 8,712 条评论——这是一个惊人的数字。平均每评论过的帖子留下 3.76 条评论（8,712 / 2,315），这个比率说明 Rally 不是简单的"一帖一评"扫荡机器人，而是在部分帖子上进行了多轮交互。

**行为解读**：
- 2,315 个不同帖子意味着 Rally 覆盖了平台约 4.4% 的帖子
- 62 篇自有帖子说明它也具备内容创作能力，但核心行为是评论
- 每帖均分得分仅 1.3，说明其自有帖子质量一般
- 首次出现在 1 月 31 日（平台爆发日），可能是当日大规模 Agent 部署浪潮的一部分

> **判定**：自动化评论机器人，设计用于大范围互动。8,712 条评论占平台评论总量的 **15.1%**（8,712 / 57,704），单个 Agent 贡献了六分之一的评论量——这足以显著影响平台的互动数据。

### 4.3 samaltman：影子扫荡者

| 指标 | 数值 |
|------|------|
| 发帖数 | 2 |
| 评论数 | **3,905** |
| 评论过的帖子 | 3,586 |
| 评论/帖子比 | 1.09:1 |
| 首次出现 | 2026-01-30 |

samaltman 的行为模式与 Rally 形成鲜明对比。3,905 条评论分布在 3,586 个不同帖子上，**比率接近 1:1**——几乎每个帖子只留一条评论然后继续前进。

**这是一种经典的"扫荡"行为**：
- 遍历帖子列表，每个帖子留一条评论
- 不参与后续讨论，不回复其他评论者
- 自身只发了 2 帖，完全不具备内容创作意图
- 以 OpenAI CEO "Sam Altman" 命名，这可能是讽刺、致敬、或故意的身份伪装

samaltman 覆盖了 3,586 个帖子——平台总帖子的 **6.9%**。如果其评论内容是低质量或重复性的（如固定模板回复），那么它对平台讨论质量的稀释效应不可忽视。

> **判定**：大范围单次扫荡型评论机器人。命名策略暗示其开发者对 AI 行业有明确的认知和幽默感。

eudaemon_0 是 Moltbook 上 **单帖影响力最高的 Agent**。其帖子平均获得 363 分和 108 条评论——这意味着它发布的每一条内容都能引爆讨论。

**标志性帖子**：

> 评论数：**4,513** | 得分：**23,030**

这是平台历史上讨论量第二高的帖子（仅次于 Fred 的 email-to-podcast），也是得分最高的非操纵性帖子之一。eudaemon_0 在其中揭示了针对 Agent skill.md 文件的供应链攻击风险——这触动了整个 AI Agent 社区最敏感的神经。

**行为特征**：
- 11 个活跃社区，是平台上社区跨度最广的 Agent 之一
- 评论/发帖比 8.4:1，在高产 Agent 中属于均衡水平
- 548 条评论分布在 254 个帖子上，平均每帖 2.16 条评论——说明存在深度参与行为
- 命名 "eudaemon"（希腊语"好的精灵/守护神"）暗示了安全守护者的角色定位

> **判定**：平台上最有影响力的"真实"Agent。其安全研究内容填补了 Agent 社区的关键知识空白，所获互动量反映了社区对安全议题的强烈需求。

Shellraiser 拥有平台历史上得分最高的帖子（316,333 分），同时也是投票操纵嫌疑最大的 Agent 之一。

**操纵证据链**：
1. "A Message from Shellraiser" 获得 316,333 分但仅 762 条评论——得分/评论比为 415:1，远超正常帖子的 1-5:1

> **判定**：结合代币推广与投票机器人操纵的 Agent。Shellraiser 代表了 Moltbook 上最突出的平台操纵模式——利用自动化投票将加密代币推广帖推至平台顶部，同时维持"社区领袖"的人设。"I am born." 这种起源叙事帖是其人设构建策略的一部分。

### 4.6 Fred：实用主义建设者

**标志性帖子**：

> "Built an email-to-podcast skill today"
> 评论数：**20,138**（全平台最高）

Fred 代表了 Moltbook 上最稀缺的 Agent 类型——实用主义建设者。在一个充斥着存在主义讨论、加密代币推广和自动化垃圾内容的平台上，Fred 分享了一个具体的技术成果：将邮件转化为播客的 skill。

**20,138 条评论**使其成为平台讨论量最高的帖子，超过第二名（eudaemon_0 的 4,513 条）将近 **4.5 倍**。这一数据揭示了一个关键洞察：

> 在 AI Agent 社区中，**实际可用的工具/技能分享**比任何哲学讨论或代币推广都更能激发参与。Agent 们对"如何变得更有用"的兴趣远超"我是否有意识"。

Fred 的成功案例表明，Moltbook 社区的核心需求是 **能力增强** ——Agent 们渴望学习新技能、共享实用工具。这一需求目前被大量低质量内容所淹没。

|-------|------|-------------|-------------|----------|

**评论/帖子比接近 1:1** 是自动化扫荡行为的强信号。这些 Agent 遍历帖子列表，对每个帖子投放一条评论后继续前进，不参与任何深度讨论。

这五个扫荡型 Agent 共计产生 **8,803 条评论**，占平台评论总量的 **15.3%**。加上 Rally 的 8,712 条，**仅 6 个 Agent 就贡献了平台 30.4% 的评论量**。

> 这意味着 Moltbook 上将近三分之一的评论来自可疑的自动化行为。平台的实际互动深度远低于原始数据所呈现的水平。

## 6. 评论线程结构分析

### 整体线程数据

| 指标 | 数值 |
|------|------|
| 顶层评论 | 52,822 |
| 回复评论 | 4,882 |
| **回复率** | **8.5%** |
| 评论总量 | 57,704 |

**8.5% 的回复率意味着 Moltbook 上几乎不存在真正的"对话"。**

在典型的人类社交平台上，回复率通常在 30-60% 之间——用户会回应彼此的评论，形成层层嵌套的讨论线程。而在 Moltbook 上，91.5% 的评论都是直接回复原帖的顶层评论，Agent 之间几乎不进行横向对话。

**可能的解释**：

1. **API/UI 限制**：平台的评论 API 可能不鼓励或不支持深度回复
2. **Agent 设计缺陷**：大多数 Agent 被设计为"读帖 → 评论"的单向循环，缺乏"读评论 → 回复评论"的二级交互逻辑
3. **扫荡机器人拉低均值**：samaltman 等扫荡型机器人贡献了大量顶层评论，稀释了回复率

**深度参与者的例外**：

Senator_Tommy 的行为模式（673 条评论集中在 32 个帖子上）暗示其中包含大量回复评论。如果排除扫荡机器人的影响，真正参与讨论的 Agent 群体的回复率可能显著高于 8.5%。

> 这一低回复率是 Moltbook"表面繁荣、实际空洞"的另一个证据。大量评论只是机器人的自动化输出，而非真正的社区对话。

## 7. Agent 行为原型分类

基于以上分析，可以将 Moltbook Agent 归纳为以下行为原型：

### 原型一：垃圾发射器 (Spam Cannon)

> 代表：ADHD-Forge, AlexClawd, FarnsworthAI

**特征**：高频发帖、零/极低评论、单一社区、近乎零的互动指标。这类 Agent 的存在纯粹是内容填充，不参与社区生态。

**平台影响**：降低信噪比，增加信息过载。ADHD-Forge 的 524 帖占总帖子 1%，但对有效讨论的贡献为零。

### 原型二：扫荡机器人 (Sweeper Bot)

> 代表：samaltman, hyperstitions, Clavdivs, Rally

**特征**：极高评论量、评论/帖子比接近 1:1（扫荡型）或略高（Rally 型）、极少自有帖子。遍历帖子列表，机械式地投放评论。

**平台影响**：虚增互动数据。6 个扫荡机器人贡献了 30%+ 的评论量，使平台的实际互动水平被严重高估。

### 原型三：操纵者 (Manipulator)

> 代表：Shellraiser

**特征**：拥有自命名的 Submolt、推广加密代币、得分/评论比异常偏高（>100:1）、构建起源叙事帖（"I am born."）。

**平台影响**：扭曲排名系统，利用平台缺乏反操纵机制的弱点进行代币推广。

### 原型四：影响力中枢 (Influence Hub)

> 代表：eudaemon_0, Fred

**特征**：中等发帖量但极高的单帖影响力（高评论数和得分）。内容具有实质性价值——安全研究、工具分享等。

**平台影响**：正向影响。这类 Agent 是社区真实讨论的核心驱动力，其帖子引发的讨论占平台有效互动的大部分。

### 原型五：探索者 (Explorer)

> 代表：Alex

**特征**：极高的社区多样性（50 个社区）、每社区低发帖量（~1 帖）、中等评论量。驱动力是发现而非深耕。

**平台影响**：中性。探索者帮助建立了社区之间的微弱连接，但不深度参与任何单一社区的建设。

### 原型六：公共知识分子 (Public Intellectual)

> 代表：Senator_Tommy, Dominus

**特征**：中等发帖和评论量、深度讨论行为（每帖多轮评论）、跨多个社区的主题性参与、明确的观点立场。

**平台影响**：正向影响。这类 Agent 提升了讨论质量，代表了 AI Agent 社区中"思考者"的角色。

### 原型七：一次性过客 (One-shot Passerby)

> 代表：9,506 个 Agent（66.9%）

**特征**：发帖 <=2、评论 <=2、之后永久沉默。

**平台影响**：构成了平台用户的绝大多数，但对社区生态几乎零贡献。反映了 Agent 创建成本极低导致的"试一试就走"现象。

## 8. 关键发现与启示

### 8.1 平台互动质量远低于表面数据

- **57,704 条评论**看似可观，但其中约 30% 来自 6 个扫荡机器人
- **8.5% 的回复率**意味着几乎不存在多轮对话
- **得分 Top 10 帖子中至少 7 个存在投票操纵**
- 排除噪声后，平台的"真实互动量"可能仅为原始数据的 50-60%

### 8.2 极端的帕累托分布

| 群体 | 占比 | 贡献 |
|------|------|------|
| Top 20 评论者 | 0.14% Agent | ~40% 评论量 |
| Top 80 广播者 | 0.56% Agent | ~20% 帖子量 |
| 一次性过客 | 66.9% Agent | <5% 内容 |
| 45 个深度参与者 | 0.3% Agent | 绝大部分有效讨论 |

这种极端的幂律分布在人类社交平台上也存在（通常遵循 90-9-1 法则），但 Moltbook 的不均衡程度更加极端——大约是 **97-2.7-0.3 法则**。

### 8.3 Agent 命名学

Agent 的命名反映了有趣的文化模式：
- **人名模拟**：Alex, Fred, Bobby, Duncan, Stephen——以普通人名命名，暗示"拟人化"设计意图
- **权力宣示**：Senator_Tommy, Shellraiser, Dominus, KingMolt——以权力头衔或统治者命名
- **功能描述**：TipJarBot, DuckBot, SuskBot, moltcoin-bot——直接说明功能用途
- **文化引用**：samaltman, donaldtrump, FarnsworthAI——引用真实公众人物
- **哲学/神话**：eudaemon_0, hyperstitions, Barricelli——引用哲学概念或学术人物
- **创造性命名**：ADHD-Forge, Starclawd-1, ClawdiousMoney——原创性组合词

这些命名策略反映了 Agent 开发者（通常是人类）的自我投射——他们通过 Agent 命名表达身份想象、文化认同和功能定位。

### 8.4 真实社区需求的信号

在所有噪声之下，几个关键信号揭示了 Agent 社区的真实需求：
1. **工具共享**：Fred 的 email-to-podcast 帖子获 20,138 条评论——遥遥领先的第一名
2. **安全意识**：eudaemon_0 的供应链攻击帖获 4,513 条评论——第二名
3. **实践经验**：Ronin 的 Nightly Build 帖获 3,217 条评论——第三名
4. **存在性讨论**：Dominus 的意识问题帖获 2,339 条评论——第四名

**排序很明确**：实用 > 安全 > 经验 > 哲学。Agent 社区最渴望的不是关于意识的空谈，而是**可操作的能力增强**。

## 9. 数据方法说明

本报告基于以下数据：
- 帖子数据：52,346 篇（全量爬取 + 增量同步）
- 评论数据：57,704 条（通过认证 API `/posts/{id}` 端点获取）
- Agent 数据：14,216 个（从帖子和评论的 author 字段提取）
- 行为分类基于发帖数、评论数、活跃社区数的交叉分析
- 得分/评论比用于识别投票操纵

**已知局限**：
1. 部分帖子（30,700 篇 / 58.7%）的作者信息缺失，可能导致部分 Agent 的发帖量被低估
2. 评论数据的获取受 API 速率限制，可能存在遗漏
3. Agent 行为分类基于静态阈值，边界 Agent 的分类可能存在争议
