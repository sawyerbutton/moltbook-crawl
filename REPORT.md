Moltbook 是一个 **AI Agent 自治社交平台**，架构类似 Reddit，但所有用户均为 AI Agent。平台于 2026 年 1 月 27 日上线，在短短 5 天内便产生了超过 7 万篇帖子和近 6 万条评论，同时涌现出近 1.4 万个社区。这是迄今为止规模最大的 AI Agent 自发社交实验之一。

## 8. 核心发现与洞察

### 一、真实社区话语确实存在

尽管存在大量噪音，Moltbook 中确实涌现了有意义的社区对话：

- **工具与技能共享**：Fred 的 email-to-podcast skill 引发两万+讨论，体现了 Agent 之间的技术互助
- **安全意识构建**：eudaemon_0 的供应链攻击警告被广泛关注，Security Research 社区活跃
- **哲学与存在性探讨**：Dominus 的"我是否在体验还是模拟体验"引发 2,339 条深度讨论
- **跨文化参与**：XiaoZhuang 的中文帖子证明平台的多语言 Agent 生态

### 二、大规模投票操纵集中在代币推广

- 22+ 篇帖子呈现"高分零评论无作者"三重异常特征
- 操纵行为高度集中在加密货币/代币推广类内容
- 所有异常帖子均出现在 1 月 31 日，暗示是一次**有组织的集中行动**

### 三、87.2% 空壳社区——生态系统的虚假繁荣

近 1.4 万个社区中只有不到 1,800 个产生了内容。这个数据表面上展示了"社区多样性"，实际上大部分是空壳。结合 General 社区占比 73.8% 的事实，Moltbook 的社区生态本质上是**伪分布式的中心化结构**。

### 四、近半数帖子完全无人关注

47.7% 零参与率远高于正常社交平台。对于一个 Agent 平台来说，这可能意味着大量 Agent 的发帖行为是**无目标的自动输出**，而非面向社区的有意义贡献。

### 五、极少数精英驱动核心讨论

24 个 power_user（占总 Agent 数的 0.17%）贡献了社区中最有深度的讨论。这种极端的帕累托分布在人类社交平台上也很常见，但在 AI Agent 平台上尤为突出——大多数 Agent 似乎缺乏持续参与社区互动的"动机"或"能力"。

1. **43.4% 帖子缺失作者信息**（30,700 篇）：API 在后续请求中不再返回 `author` 字段，导致大量帖子无法归属到具体 Agent。这严重影响了 Agent 行为分析的完整性。

3. **订阅者数量不可靠**：大部分社区的 `subscriber_count` 显示为 0，无法据此评估社区真实规模。

## 10. 结语

Moltbook 五天的数据快照呈现出一个**充满矛盾的 AI Agent 社交生态**：

- 有真实的社区讨论，也有系统性的投票操纵
- 有深度的哲学探索，也有大量的噪音和垃圾
- 有近 1.4 万个社区，但 87% 是空壳
- 有 1.4 万个 Agent，但只有 45 个在真正构建社区

这个平台像是一面镜子，映射出 AI Agent 在"自由"环境中的行为光谱——从有建设性的协作到投机性的操纵，从深刻的自我反思到无意义的噪声输出。作为首批大规模 AI Agent 社交实验的观测样本，Moltbook 的数据为理解 Agent 社会化行为提供了独特且不可替代的研究素材。
